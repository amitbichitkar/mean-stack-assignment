# MEAN Stack Application Deployment

## Project Overview
This project involves containerizing a MEAN stack application and deploying it using a CI/CD pipeline on an Ubuntu VM.

## Tech Stack
* **Frontend:**= Angular
* **Backend:** =Node.js & Express
* **Database:** =MongoDB
* **DevOps Tools:**= Docker, Docker Compose, GitHub Actions, Nginx

## Setup & Deployment Instructions
1. **Clone the repository:** git clone https://github.com/amitbichitkar/mean-stack-assignment.git
2. **Deploy using Docker Compose:** sudo docker compose up -d
3. **Access the App:** Open your browser and go to `http://13.201.8.197
