# MEAN Stack CI/CD Deployment Assignment

##   Project Overview
This project involves containerizing a MEAN stack application and deploying it using a CI/CD pipeline (GitHub Actions) on an Ubuntu VM.

##   Setup & Deployment Instructions
1. **Clone the repository:**
   `git clone https://github.com/amitbichitkar/mean-stack-assignment.git`
2. **Deploy using Docker Compose:**
   `sudo docker compose up -d`
3. **Access the App:** Open your browser and navigate to: http://13.201.8.197
